\begin{abstract} 
    This study's goal is to investigate the use of functional programming as a
    measure for reducing software defects and how it affects readability. It
    does so by looking at a specific case and introducing a REST framework
    called Cause which is completely type safe and helps creating type safe
    REST frameworks.
\end{abstract}
